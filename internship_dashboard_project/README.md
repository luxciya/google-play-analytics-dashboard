# Google Play Store Analytics Dashboard

## Project Overview

This project is an interactive dashboard built using:

- Python
- Pandas
- Plotly
- Streamlit
The dashboard analyzes Google Play Store application data using multiple advanced visualizations.

---

## Features

### Task 1 — Grouped Bar Chart

- Compare average rating and total reviews
- Top 10 categories by installs
- Time restricted visibility

### Task 2 — Choropleth Map

- Global installs visualization
- Category filtering

### Task 3 — Dual Axis Chart

- Revenue vs installs analysis
- Free vs paid apps comparison
- Interactive world map

### Task 4 — Time Series Chart

- Install trends over time
- Month-over-month growth analysis
- Multi-language category translation

### Task 5 — Bubble Chart

- App size vs rating analysis
- Bubble size represents installs
- Sentiment subjectivity filtering

### Task 6 — Stacked Area Chart

- Cumulative installs analysis
- Growth intensity highlighting
- Category translation support
## Technologies Used

- Python
- Pandas
- Plotly
- Streamlit
- NumPy
- pytz
